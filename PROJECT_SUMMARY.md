## 技术亮点

### 1. 客户端/服务器端分离
- 使用 "use client" 指令明确组件边界
- 服务器端组件处理静态内容
- 客户端组件处理交互功能

### 2. 静态生成优化
- generateStaticParams 预生成动态路由
- 构建时生成所有静态页面
- 无需服务器渲染

### 3. 主题系统
- next-themes 实现主题切换
- CSS 变量管理主题颜色
- 系统主题自动检测

### 4. 类型安全
- TypeScript 严格模式
- 组件 Props 类型定义
- 数据模型类型定义

### 5. 响应式设计
- 移动优先设计
- Tailwind 响应式类
- 灵活的网格系统
